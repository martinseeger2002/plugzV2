API Key Authentication

All API endpoints require an API key to be included in the request headers.

Include your API key in the X-API-Key header with each request.

Example:

curl -H "X-API-Key: your_api_key_here" http://blockchainplugz.com/api/v1/get_address_balance/LTC/LTCaddress123

General Errors

All error responses will be returned in the following JSON format:

{
    "status": "error",
    "message": "Error message detailing what went wrong"
}

API Endpoints

1. Get Address Balance

Endpoint: GET /api/v1/get_address_balance/<ticker>/<address>

Description: Retrieves the confirmed and unconfirmed balance for a given blockchain address.

HTTP Method: GET

Headers:

  • X-API-Key: Your API key.

Path Parameters:

  • ticker (string): The network ticker (e.g., LKY, DOGE, LTC).
  • address (string): The address to query.

Example Request:

GET /api/v1/get_address_balance/LKY/LKYaddress123

Example Response:

{
    "status": "success",
    "data": {
        "network": "LKY",
        "address": "LKYaddress123",
        "confirmed_balance": "1.234",
        "unconfirmed_balance": "0.123"
    }
}

20. Mint

Endpoint: POST /api/v1/mint

Description: Executes a mint command to create a transaction with specified parameters.

HTTP Method: POST

Headers:

  • Content-Type: application/json
  • X-API-Key: Your API key.

Request Body:

  • receiving_address (string): The address to receive the minted asset.
  • meme_type (string): The MIME type of the data.
  • hex_data (string): The data in hexadecimal format.
  • sending_address (string): The address sending the transaction.
  • privkey (string): The private key for signing the transaction.
  • utxo (string): The UTXO being spent.
  • vout (integer): The output index of the UTXO.
  • script_hex (string): The script in hexadecimal format.
  • utxo_amount (number): The amount of the UTXO.

Example Request:

POST /api/v1/mint
Content-Type: application/json
X-API-Key: your_api_key_here

{
    "receiving_address": "DCHxodkza1234567898uH6NKynmntmCNz",
    "meme_type": "text/plain",
    "hex_data": "68656c6c6f20776f726c64",
    "sending_address": "DDcp7k2e1234567890y6EyARhso2Cj",
    "privkey": "QSibKVvvbS123456789012346789ACLH2k2Hm5XYxqBXQYB",
    "utxo": "f338df94d917689e452022ffba548fe477e65dec17ac26fa1ab8a5d84b7cb895",
    "vout": 0,
    "script_hex": "76a9145cfef1b93f42326958015c4f1abddbd719c63c3a88ac",
    "utxo_amount": 421085548
}

Example Response:
